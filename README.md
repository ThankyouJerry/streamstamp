# streamstamp

유튜브 영상의 타임스탬프를 쉽게 기록하고 관리할 수 있는 Chrome 확장 프로그램입니다.

streamstamp는 이 계정의 서브 프로젝트로, 긴 영상 정리와 학습/리서치용 타임스탬프 관리에 초점을 맞춘 도구입니다.

## Project Position

- 대표작: [ClipCatcher](https://github.com/ThankyouJerry/ClipCatcher)
- 서브 프로젝트: `streamstamp`

## Main Features

- 현재 재생 위치 기준 타임스탬프 저장
- 메모와 함께 타임스탬프 관리
- 저장한 시점으로 빠르게 이동
- 마크다운 형식으로 내보내기
- 로컬 저장 기반 사용

## Use Cases

- 강의 영상 정리
- 튜토리얼 북마킹
- 리뷰 영상 메모
- 연구/학습용 타임스탬프 기록

## Privacy

모든 데이터는 로컬에 저장되며 외부 서버로 전송하지 않습니다.

## Installation

Chrome 확장 프로그램 개발자 모드에서 직접 설치해 사용할 수 있습니다.

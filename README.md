# StreamStamp Chrome Extension

YouTube 영상에서 타임스탬프를 쉽게 기록하고 관리하는 Chrome 확장 프로그램입니다.

## ✨ 주요 기능

- 📝 **간편한 타임스탬프 기록**: YouTube 영상 재생 중 언제든지 타임스탬프 추가
- 🎯 **직관적인 인터페이스**: 드래그 가능한 플로팅 버튼과 패널
- 📋 **효율적인 관리**: 모든 저장된 영상을 한눈에 확인
- 📤 **마크다운 내보내기**: 타임스탬프를 마크다운 형식으로 복사
- 🔒 **완벽한 프라이버시**: 모든 데이터는 로컬에만 저장, 서버 전송 없음

### 수동 설치
1. 이 저장소를 다운로드하거나 클론합니다
2. Chrome에서 `chrome://extensions/` 페이지를 엽니다
3. 오른쪽 상단의 "개발자 모드"를 활성화합니다
4. "압축해제된 확장 프로그램을 로드합니다"를 클릭합니다
5. `extension` 폴더를 선택합니다

## 💡 사용 방법

1. YouTube 영상 페이지로 이동
2. 오른쪽 하단의 ⏱️ 플로팅 버튼 클릭
3. 메모 입력 후 "타임스탬프 추가" 클릭
4. 저장된 타임스탬프는 확장 프로그램 아이콘에서 확인

### 주요 기능

- **타임스탬프 추가**: 현재 재생 시간과 메모를 저장
- **타임스탬프 클릭**: 해당 시간으로 바로 이동
- **개별 삭제**: 각 타임스탬프 옆의 × 버튼
- **전체 삭제**: 현재 영상의 모든 타임스탬프 삭제
- **마크다운 복사**: 타임스탬프를 마크다운 형식으로 클립보드에 복사
- **UI 위치 조정**: 플로팅 버튼과 패널을 드래그하여 원하는 위치로 이동

# StreamStamp 설치 가이드

## 설치 방법

### 방법 1: 개발자 모드로 설치 (현재 사용 중)

1. **Chrome 확장 프로그램 페이지 열기**
   - Chrome 주소창에 `chrome://extensions/` 입력

2. **개발자 모드 활성화**
   - 오른쪽 상단의 "개발자 모드" 토글 켜기

3. **확장 프로그램 로드**
   - "압축해제된 확장 프로그램을 로드합니다" 클릭
   - `/Users/hvs/.gemini/antigravity/scratch/streamstamp/extension` 폴더 선택

4. **완료!**
   - StreamStamp이 확장 프로그램 목록에 나타남
   - YouTube 페이지로 이동하여 사용 시작

---

### 방법 2: ZIP 파일로 설치

다른 사람과 공유할 때 사용하는 방법:

1. **ZIP 파일 생성**
   ```bash
   cd /Users/hvs/.gemini/antigravity/scratch/streamstamp
   zip -r streamstamp-extension.zip extension/
   ```

2. **ZIP 파일 공유**
   - 생성된 `streamstamp-extension.zip` 파일을 공유

3. **설치 (받는 사람)**
   - ZIP 파일 압축 해제
   - 위의 "방법 1" 단계 따라하기

## 사용 방법

### 1. YouTube 영상에서 타임스탬프 기록

1. YouTube 영상 페이지로 이동
2. 오른쪽 하단의 ⏱️ 플로팅 버튼 클릭
3. 메모 입력 후 "타임스탬프 추가" 클릭
4. 타임스탬프가 저장됨

### 2. 타임스탬프 관리

- **타임스탬프 클릭**: 해당 시간으로 이동
- **× 버튼**: 개별 타임스탬프 삭제
- **전체 삭제**: 현재 영상의 모든 타임스탬프 삭제
- **마크다운 복사**: 타임스탬프를 마크다운 형식으로 복사

### 3. 저장된 영상 보기

1. 확장 프로그램 아이콘 클릭
2. 모든 저장된 영상 목록 확인
3. 영상 클릭하면 YouTube 페이지로 이동
4. 영상 × 버튼으로 영상 삭제

### 4. UI 위치 조정

- **플로팅 버튼**: 드래그하여 원하는 위치로 이동
- **타임스탬프 패널**: 헤더를 드래그하여 이동
- 위치는 자동으로 저장됨

## 문제 해결

### 확장 프로그램이 작동하지 않아요
1. `chrome://extensions/`에서 StreamStamp이 활성화되어 있는지 확인
2. YouTube 페이지 새로고침 (F5)
3. 확장 프로그램 새로고침 후 다시 시도

### 플로팅 버튼이 보이지 않아요
1. YouTube 영상 페이지(`youtube.com/watch?v=...`)에 있는지 확인
2. 페이지 새로고침
3. F12 → Console에서 에러 확인

### 타임스탬프가 저장되지 않아요
1. Chrome 저장소 권한 확인
2. 개발자 도구(F12) → Console에서 에러 확인
3. 확장 프로그램 재설치

---

## 데이터 저장

- 모든 타임스탬프는 **로컬 저장소**에 저장됩니다
- 서버에 업로드되지 않으며 완전히 오프라인으로 작동합니다
- 데이터는 Chrome 프로필에 저장되므로 다른 기기와 동기화되지 않습니다

---

## 제거 방법

1. `chrome://extensions/`로 이동
2. StreamStamp 카드에서 "제거" 클릭
3. 저장된 모든 타임스탬프도 함께 삭제됩니다
